### Mục tiêu tuần 4:

* Thực hành tối ưu hệ thống AWS về bảo mật, hiệu năng và chi phí.

### Kết quả đạt được tuần 4:

* Thực hành tối ưu hệ thống AWS dựa trên các tiêu chí về bảo mật, hiệu năng và tối ưu chi phí vận hành.
* Cấu hình giám sát tài nguyên hệ thống bằng Amazon CloudWatch và Grafana; đồng thời thực hiện tự động hóa một số tác vụ quản lý bằng AWS Lambda và AWS Systems Manager.
* Thiết lập và quản lý các cơ chế bảo mật trên AWS như IAM Permission Boundary, AWS Security Hub, AWS WAF và AWS KMS nhằm tăng cường an toàn cho hệ thống.
* Triển khai ứng dụng bằng Docker trên Amazon ECS và xây dựng quy trình CI/CD với AWS CodePipeline để tự động hóa quá trình triển khai ứng dụng.
* Thực hiện tối ưu tài nguyên và chi phí sử dụng dịch vụ AWS thông qua Auto Scaling, Savings Plans, Reserved Instances và theo dõi chi phí vận hành hệ thống.
