### Week 1 Objectives

- Become familiar with AWS and its core service categories.
- Learn account, cost, and access management with IAM.
- Build a basic network and deploy an application on Amazon EC2.

### Tasks to be carried out this week:

| Day | Task | Start Date | Completion Date | Reference Material |

| 2 | **AWS Overview**<br>- Studied cloud computing and the main AWS service groups: Compute, Storage, Networking, and Database.<br>- Became familiar with the AWS Management Console, service search, and Region selection.<br>- Learned how to create and manage an AWS account and reviewed Free Tier considerations. | 04/27/2026 | 04/27/2026 | <https://cloudjourney.awsstudygroup.com/> |
| 3 | **Cost and Access Management**<br>- Explored AWS Billing, Cost Explorer, and Budgets for usage monitoring.<br>- Studied AWS support mechanisms and the Shared Responsibility Model.<br>- Practiced creating IAM users, groups, and policies, and distinguished the root account from IAM users. | 04/28/2026 | 04/28/2026 | <https://cloudjourney.awsstudygroup.com/> |

### Week 1 Results

- Learned to use the AWS Management Console, manage an account, and monitor basic costs.
- Created IAM permissions following the principle of least privilege.
- Completed a basic VPC, launched EC2, and deployed a sample application.
