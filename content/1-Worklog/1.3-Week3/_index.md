### Week 3 Objectives

- Understand the AWS migration process and migration strategies.
- Practice migrating virtual servers and databases to AWS.
- Validate and optimize workloads after migration.

- Understood the assessment, planning, execution, and validation stages of AWS migration.
- Practiced the virtual server migration workflow with VM Import/Export.
- Learned how DMS and SCT support database migration with reduced downtime.
